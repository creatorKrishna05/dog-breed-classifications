
# 🐶 Dog Breed Classification using MobileNetV2

A Deep Learning project that classifies dog breeds using **Transfer Learning (MobileNetV2)** and **TensorFlow/Keras**.

## 📌 Project Overview

This project uses a pre-trained MobileNetV2 model to identify dog breeds from images. The model is trained using TensorFlow and Keras on the Kaggle Dog Breed Identification dataset.

---

## 🚀 Features

- Image Classification using Deep Learning
- Transfer Learning with MobileNetV2
- TensorFlow Dataset Pipeline
- Data Preprocessing
- Model Training & Validation
- Model Evaluation
- Accuracy & Loss Graphs
- Predict Dog Breed from Custom Images
- Top-5 Breed Predictions
- Model Checkpoint
- TensorBoard Support
- Kaggle Submission File Generation
